# Turtlebot_3_A_star_Path_Finder

## Clone Workspace
```bash
git clone https://github.com/khuzema-h/Turtlebot_3_A_star_Path_Finder.git
```

## Build Workspace
```bash
cd ~/project3_ws
colcon build
```
## Source Bash
```bash
source install/setup.bash
```
## Lauch Turtlebot3 in Empty World
```bash
ros2 launch turtlebot3_project3 empty_world.launch.py
```
## Run Trajectory Controller
```bash
ros2 run turtlebot3_project3 trajectory_controller.py 
```
